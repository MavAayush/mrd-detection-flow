üéØProject Overview

This project implements advanced machine learning and deep learning approaches to detect Minimal Residual Disease (MRD) in flow cytometry data. MRD detection is crucial for monitoring treatment response and predicting relapse in cancer patients, particularly in hematological malignancies.

Key Innovation: We developed unsupervised anomaly detection models that learn from healthy patient data to identify cancerous cells in new patients, achieving high accuracy without requiring labeled anomalous examples during training.
